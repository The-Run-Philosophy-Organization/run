# 墨西哥永久居民获得攻略

## 简介

墨西哥绿卡的申请流程为
1. 申请临时居民签证。临时居民签证可以提供最长4年的居住期。第一年和第四年需要去墨西哥移民管理部门进行签证换发。
2. 临时居民签证转为永久居民签证。临时居民签证四年期满后可以转换为墨西哥永久居民。（即墨西哥绿卡）
